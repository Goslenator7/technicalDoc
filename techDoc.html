About Python

Python is powerful... and fast; plays well with others; runs everywhere; is friendly & easy to learn; is Open.

Python can be easy to pick up whether you're a first time programmer or you're experienced with other languages.

Python offers many choices for web development:
  • Frameworks such as Django and Pyramid.
  • Micro-frameworks such as Flask and Bottle.
  • Advanced content management systems such as Plone and django CMS.
Python Data Types

Python has a number of recognised data types:

  • Text: str
  • Numeric: int, float, complex
  • Sequence: list, tuple , range
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: byte, bytearray, memoryview
Python Variables

To initialise a variable in Python, do this:

String: x = "This is a string"

Numeric: x = 20

Sequence: x = ["apple", "banana", "cherry"]

Mapping: x = {"age" : 28, "name" : "Clive"}

Boolean: x = True

Python Functions

To create functions in Python, you first have to define them:

def myFunction(): print ("Hello world")

Now that the function is defined, you call the function somewhere in your code:

myFuction()
